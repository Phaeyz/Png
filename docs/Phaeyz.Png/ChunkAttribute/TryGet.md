Gets the [`ChunkAttribute`](../ChunkAttribute.md) of a chunk class type.

```csharp
public static bool TryGet(Type chunkClassType, out ChunkAttribute chunkAttribute)
```

| parameter | description |
| --- | --- |
| chunkClassType | The chunk class type. |
| chunkAttribute | If the method returns `true` this will receive the chunk attribute. Otherwise it will receive `null`. |

## Return Value

Returns `true` if the chunk class type is valid and the *chunkAttribute* received the attribute; `false` otherwise.

## Remarks

This function always returns `false` if the chunk class type does not derive from [`Chunk`](../Chunk.md) or if the chunk class type does not have a public default parameterless constructor.
