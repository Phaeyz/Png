# ChunkDefinitions.Add method (1 of 2)

Adds a mappings for a chunk type to a chunk class.

```csharp
public void Add(Type chunkClassType, bool overrideExistingMappings = false)
```

| parameter | description |
| --- | --- |
| chunkClassType | The chunk type to add a mapping for. |
| overrideExistingMappings | If `true` and there is an existing mapping for the chunk type to a chunk class, it will be removed and replaced. If `false`, an exception will be thrown if a mapping for the chunk type already exists. The default value is `false`. |

## Exceptions

| exception | condition |
| --- | --- |
| InvalidOperationException | The current object is read-only. |
| ArgumentException | A configuration issue occurred with the chunk class, or a mapping for the chunk type already exists. |
