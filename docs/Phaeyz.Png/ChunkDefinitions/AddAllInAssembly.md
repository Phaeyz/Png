# ChunkDefinitions.AddAllInAssembly method

Adds all eligible mappings for chunk types to chunk classes in an assembly.

```csharp
public void AddAllInAssembly(Assembly assembly, bool overrideExistingMappings = false)
```

| parameter | description |
| --- | --- |
| assembly | The assembly containing the chunk classes to add mappings for. |
| overrideExistingMappings | If `true` and there is an existing mapping for a chunk type to a chunk class, it will be removed and replaced. If `false`, an exception will be thrown if a mapping for a chunk type already exists. The default value is `false`. |

## Exceptions

| exception | condition |
| --- | --- |
| InvalidOperationException | The current object is read-only. |
| ArgumentException | A mapping for a chunk type already exists. |
