# PngMetadata.ReadFromStreamAsync method

Read a PNG metadata from the stream.

```csharp
public static ValueTask<PngMetadata?> ReadFromStreamAsync(MarshalStream stream, 
    ChunkDefinitions? pngDefinitions = null, CancellationToken cancellationToken = default)
```

| parameter | description |
| --- | --- |
| stream | The stream to read the PNG metadata from. |
| pngDefinitions | Optionally chunk definitions which maps chunk keys to chunk class types. Specify `null` to use the default chunk definitions. |
| cancellationToken | A cancellation token which may be used to cancel the operation. |

## Return Value

Returns a [`PngMetadata`](../PngMetadata.md) if the stream contains valid PNG metadata, otherwise `null` if a PNG signature could not be detected.

## Remarks

It is expected the last chunk be image-end.
