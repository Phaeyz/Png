# PngMetadata.ReadAllFromStreamAsync method

Reads all PNG metadatas from the stream.

```csharp
public static ValueTask<List<PngMetadata>> ReadAllFromStreamAsync(MarshalStream stream, 
    ChunkDefinitions? chunkDefinitions = null, CancellationToken cancellationToken = default)
```

| parameter | description |
| --- | --- |
| stream | The stream to read the PNG metadata from. |
| chunkDefinitions | Optionally chunk definitions which maps chunk types to chunk class types. Specify `null` to use the default chunk definitions. |
| cancellationToken | A cancellation token which may be used to cancel the operation. |

## Return Value

Returns a list of [`PngMetadata`](../PngMetadata.md) objects based on what has been discovered in the stream.

## Remarks

If the stream contains multiple back-to-back PNGs, this method will return all of them.
