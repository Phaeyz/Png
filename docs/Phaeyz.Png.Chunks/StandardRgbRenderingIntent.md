# StandardRgbRenderingIntent enumeration

The specified rendering intent defined by the International Color Consortium [ICC] or [ICC-2].

```csharp
public enum StandardRgbRenderingIntent : byte
```

## Values

| name | value | description |
| --- | --- | --- |
| Perceptual | `0` | For images preferring good adaptation to the output device gamut at the expense of colorimetric accuracy, such as photographs. |
| RelativeColorimetric | `1` | For images requiring color appearance matching (relative to the output device white point), such as logos. |
| Saturation | `2` | For images preferring preservation of saturation at the expense of hue and lightness, such as charts and graphs. |
| AbsoluteColorimetric | `3` | For images requiring preservation of absolute colorimetry, such as previews of images destined for a different output device (proofs). |
